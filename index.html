키워드를 입력하고, 키워드별 동의어를 입력하면 검색식이 자동으로 완성됩니다. 즉, 단어에만 집중할 수 있습니다.

모든 단어는 태그 처리되어 가독성이 좋아졌을 뿐만 아니라, 마우스 오른쪽 버튼으로 제거할 수 있고, 왼쪽 버튼으로 수정할 수 있습니다.

검색식도 직접 입력할 필요가 없고, '특허 검색' 버튼을 누르면, 자동으로 선택한 사이트로 이동하여 검색 결과를 표시합니다.

키워드별로 검색할 필드를 지정할 수 있고, 이것은 선택 즉시 검색식에 자동으로 반영됩니다.

검색 내용은 자동으로 저장되고, 각 기록에 대하여 평가하고 메모할 수 있습니다. 또한 메인 화면에 복구할 수도 있습니다.
